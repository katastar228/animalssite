#  Энциклопедия домашних питомцев

Вариант 2: "Мир животных"

##  О проекте
Веб-приложение на Flask с использованием блюпринтов. 
Энциклопедия с информацией о кошках и собаках.

##  Запуск
1. Установите Flask: pip install flask
2. Запустите python main.py или animalssite.bat
3. Откройте http://127.0.0.1:5000

##  Структура
- main.py - главный файл
- routes/ - блюпринты
- templates/ - HTML шаблоны
- animalssite.bat - файл для запуска

##  Страницы
- Главная (/)
- Кошки (/cats/, /cats/breeds, /cats/care)
- Собаки (/dogs/, /dogs/breeds, /dogs/training)
